Abstract

Ancient stele inscriptions preserve cultural memory and calligraphic aesthetics, but environmental and human factors create severe information loss in rubbing images. This work treats inscription restoration as a multimodal benchmark task.

The paper constructs CIRI, a Chinese Inscription Rubbing Image dataset with diverse calligraphy styles, character structures, and degradation forms. It also introduces a synthesis approach for intact-degraded pairs and a baseline pipeline that combines global restoration, text rectification, and local refinement.

Benchmark Focus

  • Real inscription rubbing images from electronic stele collections.
  • Degradation forms including scatter noise, spalling, grid etching, and irregular absence.
  • Evaluation of both visual presentation and textual integrity.

Global-Local Restoration Pipeline

The baseline pipeline restores a degraded rubbing through three modular stages. Each stage is trained independently so the system can preserve binary rubbing structure while injecting textual and glyph knowledge.

  • GRM: a discrete diffusion global restoration module for binary rubbing images.
  • TRM: text rectification using character detection, recognition, ordering, and corpus retrieval.
  • LRM: local character refinement guided by restored context and rendered character structure.
